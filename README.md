## 📖 Tentang

<div align="center">
<img src="https://capsule-render.vercel.app/api?type=soft&color=gradient&height=2&width=80%&section=spacer" alt="divider" />
</div>

> **LearnHub** adalah proyek web statis yang dibangun **murni untuk keperluan belajar dan edukasi**. Proyek ini menjadi wadah latihan untuk memahami konsep-konsep pengembangan web modern — mulai dari struktur HTML semantik, styling dengan CSS/Tailwind, hingga interaktivitas menggunakan JavaScript.

Semua konten yang ada di dalam proyek ini **hanyalah data dummy** dan tidak merepresentasikan produk, layanan, atau entitas nyata manapun.

## 🛠️ Instalasi

<div align="center">
<img src="https://capsule-render.vercel.app/api?type=soft&color=gradient&height=2&width=80%&section=spacer" alt="divider" />
</div>

### Prasyarat

<img src="https://img.shields.io/badge/Node.js-v18%2B-339933?style=flat-square&logo=node.js&logoColor=white" alt="Node" />
<img src="https://img.shields.io/badge/Git-Latest-F05032?style=flat-square&logo=git&logoColor=white" alt="Git" />

### Langkah-langkah

```bash
# 1. Clone repositori ini
git clone https://github.com/username/learnhub.git

# 2. Masuk ke direktori proyek
cd learnhub

# 3. Buka file index.html di browser
# Atau gunakan Live Server (VS Code Extension)
code .

# 4. Klik "Go Live" di status bar VS Code
```

<details>
<summary>🔧 Opsi: Jalankan dengan CLI sederhana</summary>

```bash
# Jika kamu punya Python terinstall
python -m http.server 8000

# Atau dengan Node.js (npx)
npx serve .

# Buka browser di http://localhost:8000
```

</details>
