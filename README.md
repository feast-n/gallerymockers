## 📖 Tentang

<div align="center">
<img src="https://capsule-render.vercel.app/api?type=soft&color=gradient&height=2&width=80%&section=spacer" />
</div>

> **LearnHub** adalah proyek web statis yang dibangun murni untuk keperluan **belajar dan edukasi**. Proyek ini menjadi wadah latihan untuk memahami konsep-konsep pengembangan web — mulai dari struktur HTML semantik, styling CSS, hingga interaktivitas JavaScript.
>
> Semua konten yang ada di dalam proyek ini **hanyalah data dummy** dan tidak merepresentasikan produk, layanan, atau entitas nyata manapun.

## 🛠️ Instalasi

<div align="center">
<img src="https://capsule-render.vercel.app/api?type=soft&color=gradient&height=2&width=80%&section=spacer" />
</div>

```bash
# 1. Clone repositori
git clone https://github.com/feast-n/learnhub.git

# 2. Masuk ke direktori
cd learnhub

# 3. Buka index.html di browser
# Atau gunakan Live Server di VS Code
```

<details>
<summary>🔧 Alternatif: Jalankan via CLI</summary>

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

</details>
